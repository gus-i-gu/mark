# Supporting Markei

Markei is an independent project under active development. Support can help sustain development time, testing environments, distribution costs, documentation, and future privacy-preserving infrastructure.

When GitHub Sponsors is active for the repository owner, GitHub may display a **Sponsor** button using `.github/FUNDING.yml`.

## What sponsorship means

Sponsorship is voluntary support for the project's continued development. It does not purchase:

- access to household or user data;
- priority over security or privacy boundaries;
- ownership of project decisions;
- guaranteed delivery dates or private support;
- influence over contributed agency datasets.

## Non-financial support

Helpful forms of support also include:

- careful bug reports with reproducible evidence;
- testing on supported environments;
- documentation corrections;
- bounded design discussion;
- privacy and accessibility review;
- sharing the project with people who may find its household-economic perspective useful.

For contribution rules, see `CONTRIBUTING.md`.
